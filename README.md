# Blog de Platzi - Curso de Git y GitHub

Este es un proyecto educativo realizado como parte del curso de Git y GitHub de Platzi. El objetivo es aprender y aplicar los conceptos fundamentales del control de versiones.

## Características

- Estructura HTML5 semántica.
- Diseño responsivo y moderno con CSS.
- Flujo de trabajo basado en Git.

## Cómo ver el proyecto

Puedes abrir el archivo `index.html` en cualquier navegador web moderno.
